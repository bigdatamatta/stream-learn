strlearn.ensembles.ChunkBasedEnsemble¶

class strlearn.ensembles.ChunkBasedEnsemble(ensemble_size=5)¶

Chunk based ensemble classifier.

Ensemble classifier composed of estimators trained on the fixed number of previously seen data chunks.

ensemble_sizeinteger, optional (default=5)

The maximum number of estimators trained using consecutive data chunks and maintained in the ensemble.

ensemble_list of classifiers

The collection of fitted sub-estimators.

classes_array-like, shape (n_classes, )

The class labels.

>>> import strlearn as sl
>>> stream = sl.streams.StreamGenerator()
>>> clf = sl.ensembles.ChunkBasedEnsemble()
>>> evaluator = sl.evaluators.TestThenTrainEvaluator()
>>> evaluator.process(clf, stream)
>>> print(evaluator.scores_)
